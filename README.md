# X-12 Series Project Website

这是 X-12 Series 项目的官方网站，展示了包括 XERUM OS、音乐作品和软件应用等内容。该网站是 LKKC 校际艺术比赛的参赛作品之一。

## 功能特点

- 现代化的蓝色主题设计
- 响应式布局，支持各种设备
- 流畅的动画效果
- 项目展示卡片
- 详细的关于页面

## 技术栈

- HTML5
- CSS3
- JavaScript
- GSAP (动画库)
